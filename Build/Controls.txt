-- Info --

There are 3 types of units
	- Melee (Black)
		- Health: 100
		- AP: 10
	- Ranged (Green)
		- Health: 50
		- AP: 7
	- Magic (Purple)
		- Health: 75
		- AP: 5

- There are 8 enemies
- enemies have less health
- fixed the accidental movement when clicking
- blockout map added

-- Controls --

Movement:
Left Mouse Button
	- Select Team Characters
	- Select tile to move to
	- Attack enemy (When item is active)
Left Mouse Button (Drag)
	- Rotate Camera
Middle Mouse Button (Drag)
	- Move Camera
Scroll Up
	- Zoom in
Srcoll Down
	- Zoom out

Attacking:
Note: Both attack have a range and only work if there is line of
sight between selected character and target. Units that can be attacked
will be outlined.

Q
	- Activate equipted weapon (Range 5)
W
	- Activate ability (Range 20)

UI:
Console - This displays any errors or logs that happen in game
	  and will output them to a file called DebugLog.txt
	  that is located in the data file next to the .exe
Attack
	- When click will perform same action as 'Q' but will also outline
	  enemys that are in range when hovering over the button
Ability
	- When click will perform same action as 'W' but will also outline
	  enemys that are in range when hovering over the button


Misc:
Space
	- Reset action points
ESC
	- Close the program
